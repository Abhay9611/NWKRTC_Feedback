NWKRTC Feedback System

A QR Code-based Feedback Collection System for Bus Stations

QR Code Generation

Generate unique QR codes for each bus station. Users can scan these codes to provide feedback about their experience.

Comprehensive Feedback

Collect detailed feedback about:

  • Toilet Cleanliness
  • Bus Stand Cleanliness
  • Drinking Water Facilities
  • Toilet Usage Fees
  • Overall Experience

Bilingual Support

The system supports both English and Kannada languages for better accessibility.
